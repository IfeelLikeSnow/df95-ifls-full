DF95 Full v10 IDM Edition
=========================

Dieses Paket enthält:
- DF95_Metadata_Core.lua v2 (erweiterte IDM/Tape/Granular-Tags)
- DF95_IDM_FXChain_Generator_Pro.lua v2
- DF95_Metadata_Report_IDM.lua
- DF95_Dashboard_Center.lua (mit IDM & Metadata Sektion)
- AutoInstaller, Validator Pro, Hubs, Icons und Toolbars aus v7

Installation:
1. ZIP entpacken.
2. Inhalt in das REAPER-Resource-Verzeichnis kopieren
   (in REAPER: Options -> Show REAPER resource path...).
3. In der Action List folgende Skripte laden:
   - DF95_Dashboard_Center.lua
   - optional DF95_IDM_FXChain_Generator_Pro.lua
   - optional DF95_Metadata_Report_IDM.lua
4. Dashboard starten und im Menü "IDM & Metadata" nutzen.

Voraussetzung:
- REAPER 6/7 (64-bit)
- Für automatisches Öffnen von Reports: SWS Extension (optional).
