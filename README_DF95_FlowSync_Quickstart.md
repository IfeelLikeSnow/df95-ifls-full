# DF95 FlowSync – Quickstart

**Signalfluss:** Recording → Explode → FX Bus → Coloring → Master → QA

**Kernbuttons:**
- Explode AutoBus: Routing & Bus‑Setup
- Mic FX: Mikrofonspezifische Chains
- FX Bus / Seed: IDM/Glitch Chains + deterministisches Reseeding
- Coloring Bus + Bias: Tonfärbung (Artists/Neutral/User)
- Master Tools: SmartCeiling, LUFS GainMatch, Parallel FX Presets, Validator

**Lautheit:**
- Ziel: Loops/Samples −0.3 dBTP / −12..−14 LUFS
- Streaming: −1.0 dBTP (SmartCeiling Preset)

**Overlay:**
- Click „Bias“/„Ceiling“/„GainMatch“ → öffnet passende Menüs
- PIN für dauerhaft, sonst Auto‑Hide (8 s)
