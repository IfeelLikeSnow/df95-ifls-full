Built: 2025-11-11 20:04:43

Installation
1) Entpacke das ZIP in deinen REAPER‑Ressourcenpfad (Options → Show REAPER resource path).
2) Starte: Scripts/IFLS/DF95/DF95_AutoInstall_ThemeAdaptive.lua
3) In REAPER: Options → Customize menus/toolbars → Main toolbar → Import:
   Menus/DF95_MainToolbar_FlowErgo_Pro.ReaperMenuSet → Apply
4) Toolbar‑Icons zuweisen (Customize → Button → Change Icon), Ordner:
   Data/toolbar_icons/DF95/

Hinweise
- Farbschema: Balanced Studio (ruhig, kreativitätsfördernd, nicht grell)
- UX‑Profil: Scripts/IFLS/DF95/DF95_UX_Profile.json
- Icons: PNG (32px) + SVG‑Beispiele
- Spacer‑Empfehlung per Script: DF95_UX_Helper_Spacers.lua
