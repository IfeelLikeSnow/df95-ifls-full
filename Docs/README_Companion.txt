
DF95 UI – FlowErgo Creative v1 (Balanced Studio) – THEME COMPANION
Built: 2025-11-11 20:06:33

Enthalten:
- Theme/DF95_BalancedStudio.ReaperTheme
- Theme/toolbar_bg.png (dezente Toolbar-Hintergrundgrafik)
- Scripts/IFLS/DF95/DF95_AutoIcon_Assign.lua (schreibt ICON-Zeilen in dein MenuSet)
- Menus/DF95_MainToolbar_FlowErgo_Pro.ReaperMenuSet (Bequemlichkeit)
- Data/toolbar_icons/DF95/ (Icon-Dateien)

Installation:
1) Entpacke das ZIP in deinen REAPER-Ressourcenpfad.
2) Theme wählen: Options → Themes → Import DF95_BalancedStudio.ReaperTheme
   (Die Datei 'toolbar_bg.png' wird automatisch gefunden, wenn das Theme entpackt im selben Ordner liegt.)
3) Toolbar importieren oder aktualisieren:
   - Auto-Install laufen lassen (DF95_AutoInstall_ThemeAdaptive.lua) ODER
   - Menus → DF95_MainToolbar_FlowErgo_Pro.ReaperMenuSet manuell importieren
4) Icons automatisch zuordnen:
   Actions → Run: DF95_AutoIcon_Assign.lua → Toolbar danach re-importieren und Apply.

Hinweise:
- DF95_BalancedStudio.ReaperTheme ist "leichtgewichtig": nur dezente Farben + Toolbar-Hintergrund.
- Du kannst jederzeit zurück zum bisherigen Theme wechseln; Icons & Toolbar bleiben nutzbar.
