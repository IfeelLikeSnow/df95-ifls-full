3) Options → Themes → Import → DF95_BalancedStudio.ReaperThemeZip → auswählen
4) Options → Customize menus/toolbars → Main toolbar → Import:
   Menus/DF95_MainToolbar_FlowErgo_Pro.ReaperMenuSet → Apply
5) Actions → Run: DF95_FlowStudio_Finalize.lua  (ThemeSync + Icons + Toolbar-Fenster)
6) LiveCheck-Button in Toolbar ausführen → Validator prüfen
