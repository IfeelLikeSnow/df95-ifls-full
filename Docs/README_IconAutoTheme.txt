
DF95 UI – Icon Auto-Theme v1
Built: 2025-11-11 20:11:09

Enthalten:
- Scripts/IFLS/DF95/DF95_AutoIcon_Assign_ThemeAware.lua
  (ordnet Icons automatisch zu und wählt je nach Theme Helligkeit 'dark' oder 'light')
- Data/toolbar_icons/DF95/dark/*.png
- Data/toolbar_icons/DF95/light/*.png

Benutzung:
1) Stelle sicher, dass dein MenuSet bereits unter REAPER\Menus liegt.
2) Actions → Run: DF95_AutoIcon_Assign_ThemeAware.lua
3) Danach Toolbar re-importieren und Apply.

Hinweis:
- Helligkeit wird grob aus dem Theme-Namen geschätzt (dark/balanced/hydra/imperial → dunkel).
- Du kannst jederzeit manuell das Set ändern, indem du die ICON-Zeilen im MenuSet anpasst.
